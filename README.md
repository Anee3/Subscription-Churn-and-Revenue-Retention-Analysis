
📊 Subscription Churn & Revenue Retention Analysis

📌 Project Overview

This project analyzes customer subscription data to identify churn patterns, revenue trends, and retention insights. The goal is to understand why customers leave and how subscription plans impact revenue.

🎯 Objectives

1) Analyze customer churn behaviour

2) Identify high-risk subscription plans

3) Measure revenue generated by each plan

4) Calculate revenue lost due to churn

5) Present insights using data visualizations and dashboards

🛠 Tools & Technologies

-- Python

-- Pandas

-- Matplotlib

-- Seaborn

-- SQL

Google Sheets (Dashboard & Pivot Tables)

📈 Dashboard

A visual dashboard was created in Google Sheets to present key metrics including:

-- Customer churn distribution
-- Revenue by subscription plan
-- Revenue loss due to churn
-- Customer retention patterns


🔍 Key Insights

-- Month-to-month subscriptions show the highest churn rate

-- Long-term subscription plans retain customers more effectively

-- A significant portion of revenue loss comes from short-term subscribers

-- Improving retention strategies for monthly plans can reduce churn
